# 四柱推命 Chrome拡張機能 ユーザーガイド

この拡張機能は、ブラウザのサイドパネルで手軽に四柱推命の命式を作成できるツールです。

## インストール方法

1. このリポジトリを適当なフォルダにダウンロード（解凍）します。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力して移動します。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、ダウンロードしたフォルダの中にある `chrome-extension` フォルダを選択します。
5. ツールバーのパズルアイコンから「四柱推命」拡張機能をピン留めすると便利です。

## 基本的な使い方

### 1. 起動する
ブラウザ右上の拡張機能アイコンをクリックし、「サイドパネルを開く」を選択するか、サイドパネルメニューから「四柱推命」を選択します。

### 2. 情報を入力する
- **生年月日**: 西暦で入力します（例: 1990年1月1日）。
- **時刻**: 出生時刻がわかる場合は入力します。不明な場合は12:00などで計算されますが、時柱が不正確になる可能性があります。
- **性別**: 大運の順行・逆行判定に使用します。

**便利な入力方法**:
- **クリップボード貼り付け**: テキストエリアに「1990/01/01 12:00 男性」のようなテキストを貼り付けると、自動的に各フィールドに入力されます。

### 3. 計算する
「命式を計算する」ボタンをクリックします。
下にスクロールすると、以下の情報が表示されます。
- **命式表**: 年干支、月干支、日干支、時干支
- **通変星・蔵干・十二運**: 各柱の詳細情報
- **大運表**: 0〜100歳までの10年ごとの運勢

### 4. 結果を保存する
- **PNG画像として保存**: 結果エリアの左上にある「画像として保存」ボタンを押すと、命式部分がPNG画像としてダウンロードされます。SNSでの共有や鑑定書の作成に便利です。

## 注意事項
- 本ツールは日本の標準時（JST）を前提としています。海外生まれの場合は時差を考慮して入力してください。
- 1900年〜2100年頃までの節気データに対応しています。

## トラブルシューティング
- 計算されない場合: 入力形式が正しいか確認してください（未来の日付や存在しない日付など）。
- エラーが表示される場合: いったん拡張機能を再読み込み（リロード）してみてください。
