# Calculation Algorithms

本アプリケーションで使用されている四柱推命の計算アルゴリズムについての解説です。

## 1. 暦と節入り

本システムは、太陽の黄経に基づく二十四節気（定気法）を使用しています。
国立天文台などが公開している計算式またはデータに基づき、分単位で節入り時刻を判定しています。

- **年の区切り**: 立春（2月4日頃）の節入り時刻
- **月の区切り**: 各月の節入り時刻（啓蟄、清明、...）

### 年柱の計算
1. 入力された日時が、その年の「立春」の節入り時刻より前か後かを判定します。
2. 立春前であれば前年の干支、立春後であればその年の干支を採用します。
   - 例: 2024年2月1日は、まだ立春前なので2023年（癸卯）として扱われます。

### 月柱の計算
1. 入力された日時が、どの節気に入っているかを判定し、月支（寅、卯...）を決定します。
2. 年干と月支から、五虎遁年起月訣（年干から月干を求める法則）を用いて月干を決定します。

## 2. 日柱と時柱

### 日柱
- 1900年1月1日（甲戌）を基準日とし、経過日数から60干支を算出しています。
- 日柱の変わり目は午前0時としています（夜子時説を採用せず、23時〜0時も翌日扱いとはせず、あくまで時柱のみの変化とみなすのが一般的ですが、本システムでは時柱計算において23時以降を翌日の早子時として扱うかどうかは設定によりますが、基本実装では23時で日が変わるわけではありません。ただし時柱算定では23-01時を子の刻として扱います）。
  - ※実装詳細: `calculateHourPillar` では `(hour + 1) / 2` で支を求めており、23時は翌日の0時と同じ「子」の区画に入ります。日干への影響はありません。

### 時柱
- 1日を12の「刻（2時間）」に分けます。
- 23:00 〜 00:59 を「子（ね）」の刻とします。
- 日干から、五虎遁日起時訣（日干から時干を求める法則）を用いて時干を決定します。

## 3. 大運（Taiun / Great Fortune）

10年ごとの運勢サイクルを計算します。

### 順行・逆行の判定
- **陽男陰女**（陽の年に生まれた男性、陰の年に生まれた女性）: **順行**
- **陰男陽女**（陰の年に生まれた男性、陽の年に生まれた女性）: **逆行**
- 陽の年: 天干が甲・丙・戊・庚・壬
- 陰の年: 天干が乙・丁・己・辛・癸

### 開始年齢（立運）の計算
- **順行**: 生まれた日から「次の節入り」までの日数を計算し、3で割ります。
- **逆行**: 生まれた日から「前の節入り」までの日数を計算し、3で割ります。
- 端数は四捨五入します（例: 2.6 -> 3歳運）。

## 4. 蔵干と通変星

- **蔵干**: 各地支に含まれる干（本気・中気・余気）のうち、主となるものを採用しています（本システムでは簡易化のため、月律分野蔵干の詳細な日数計算までは行わず、主要な蔵干を表示します）。
- **通変星**: 日干（自分）と他の干との五行関係（生・剋・比）および陰陽の関係から算出します。

## 5. 十二運

- 日干から見た各支のエネルギー状態（長生、沐浴...）を表組みデータから参照します。
