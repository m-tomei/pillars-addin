================================================================================
主要設計情報
================================================================================

アーキテクチャ: 4層構成 (Presentation → Application → Domain → Data)
メインクラス: 8つ
  - FortuneCalculator (四柱計算)
  - GreatFortuneCalculator (大運計算)
  - JuuniunCalculator (十二運計算)
  - TsuuhenCalculator (通変星計算) 【新規】
  - DataLoader, InputParser, ResultRenderer, ImageExporter

計算エンジン:
  年柱: (year - 1864) % 60
  月柱: 節入り時刻 + 五虎遁
  日柱: (days_from_1900-01-01 + 10) % 60
  時柱: 23:00境界 + 五虎遁

パフォーマンス目標:
  - アプリ起動: < 1.5秒
  - 四柱計算: < 300ms
  - 結果表示: < 200ms
  - PNG出力: < 2秒

実装フェーズ: 5フェーズ (合計6-8週間)
  Phase 1: 基盤構築 (1-2週)
  Phase 2: 計算エンジン (2-3週)
  Phase 3: UI実装 (2週)
  Phase 4: 最適化 (1-2週)
  Phase 5: テスト・ドキュメント (1-2週)
